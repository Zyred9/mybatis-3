* # 中文注释版 Mybatis-3

  ### **一、下载安装**

  1、下载地址： https://github.com/Zyred9/mybatis-3

  2、maven 执行安装到本地仓库，源码中版本是 3.5.4-snapshot

  ```bash
  mvn install
  ```

  3、执行编译

  ```text
  _install.bat
  ```

  4、清除编译

  ```text
   _cleanup.bat
  ```

  

  如果上述步骤失败，那么直接将项目路径下的 `mybatis-3.5.4-snapshot.jar` 放入 yourMavenPath/org/mybatis/mybatis/3.5.4-snapshot 目录下。
